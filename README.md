# first-pr-practice

A small practice project for learning the GitHub pull request workflow.

## What is a pull request?

A pull request (PR) lets you propose changes to a repository. Someone reviews
your changes, and once approved, they get merged into the main branch.

## How to contibute

1. Fork or clone the repo
2. Create a new branch
3. Make your change
4. Open a pull request
